
# 🎓 Student Performance Analysis using python

## 📌 Project Overview

This project analyzes student performance using Python to identify how different factors such as weekly self-study hours, attendance percentage, and class participation influence students' total scores and grades.

The analysis includes data cleaning, exploratory data analysis (EDA), statistical summaries, visualizations, and key insights to better understand academic performance.

---

## 📂 Dataset

The dataset contains the following features:

- Student ID
- Weekly Self Study Hours
- Attendance Percentage
- Class Participation
- Total Score
- Grade

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Analysis Performed

- Loaded and inspected the dataset
- Checked dataset information and summary statistics
- Checked for missing values
- Identified students with highest scores
- Identified students with lowest scores
- Analyzed study hours, attendance, and class participation
- Created multiple data visualizations
- Generated useful academic insights

---

## 📈 Visualizations

- Study Hours vs Total Score
- Attendance Percentage vs Total Score
- Class Participation vs Total Score
- Correlation Heatmap
- Grade Distribution

---

## 🔍 Key Findings

- Students who study more hours generally achieve higher scores.
- Higher attendance is associated with better academic performance.
- Active class participation positively impacts total scores.
- Students with very low study hours tend to receive lower grades.
